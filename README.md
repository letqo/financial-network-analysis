# Financial Network Analysis & Systemic Risk

Analyzing interdependencies between S&P 500 stocks and cryptocurrencies 
using network science to model systemic risk and extract trading signals.

## Methodology
- Correlation matrix across 55 assets (2019–2024)
- Network construction using filtered correlation graphs
- Centrality measures to identify systemic nodes
- Community detection to extract sector clusters
- Centrality-based trading signal with backtesting

## Tools
Python, pandas, numpy, networkx, matplotlib, seaborn
